# Notes App (Java File I/O)

## Description
A simple CLI-based notes manager using Java File I/O. It allows users to write and read notes stored in a text file.

## Features
- Add Notes
- View Notes

## Technologies Used
- Java
- FileWriter
- BufferedReader

## How to Run
Run using IntelliJ or:
javac NotesApp.java
java NotesApp

## Concepts Covered
- File handling
- Exception handling
- User input
